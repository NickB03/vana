```python

    name="VANA",

    tools=[hotel_tool, flight_tool, payment_tool, travel_orchestrator_tool]
)
```

#### **3. Sequential Pipeline Pattern Implementation**
```python
# Development workflow pipeline
code_generator = LlmAgent(name="CodeGenerator", instruction="Generate code based on requirements.", output_key="generated_code")
code_reviewer = LlmAgent(name="CodeReviewer", instruction="Review code from state['generated_code'] for quality.", output_key="review_status")
test_generator = LlmAgent(name="TestGenerator", instruction="Generate tests for code if review_status is 'approved'.", output_key="test_code")

development_pipeline = SequentialAgent(
    name="DevelopmentPipeline",
    sub_agents=[code_generator, code_reviewer, test_generator]
)
```

#### **4. Parallel Fan-Out/Gather Pattern Implementation**
```python
# Parallel research gathering
research_agent_1 = LlmAgent(name="WebResearcher", instruction="Research topic from web sources.", output_key="web_research")
research_agent_2 = LlmAgent(name="DatabaseResearcher", instruction="Research topic from databases.", output_key="db_research")

parallel_research = ParallelAgent(
    name="ParallelResearch",
    sub_agents=[research_agent_1, research_agent_2]
)

research_synthesizer = LlmAgent(
    name="ResearchSynthesizer",
    instruction="Combine results from state['web_research'] and state['db_research']."
)

research_workflow = SequentialAgent(
    name="ResearchWorkflow",
    sub_agents=[parallel_research, research_synthesizer]
)
```
