## Overview

The integrations documentation provides information about how VANA integrates with external systems and services. Each integration has its own section with setup instructions, configuration options, and usage examples.

## Key Integrations

- **n8n Workflows**: Integration with n8n for workflow automation
- **MCP Integration**: Integration with Model Context Protocol for memory management
- **Vertex AI Integration**: Integration with Google Cloud Vertex AI services
- **Agent Engine Integration**: Integration with Google's Agent Engine
