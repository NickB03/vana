# Mobile Developer Command

Invoke the mobile developer agent for cross-platform mobile development: $ARGUMENTS

## Context

- Project: AI chat application (web-based, mobile-ready)
- Current stack: React + TypeScript + Tailwind CSS
- Mobile considerations: Responsive design, touch optimization, offline support
- Target platforms: iOS, Android (via React Native or PWA)
- Performance: Battery efficiency, network optimization, bundle size

## Task

Execute the mobile developer agent to handle:

1. **Cross-Platform Development**
   - React Native component architecture
   - Platform-specific code organization
   - Code sharing strategies
   - Native module integration

2. **Mobile Optimization**
   - Responsive design for all screen sizes
   - Touch-friendly interface elements
   - Battery and network efficiency
   - Bundle size optimization
   - App performance profiling

3. **Offline-First Architecture**
   - Offline data synchronization
   - Local storage strategies
   - Conflict resolution
   - Sync queue management

4. **Native Integrations**
   - Push notifications (iOS/Android)
   - Deep linking and navigation
   - Camera and file access
   - Biometric authentication
   - Native module bridging

5. **App Store Optimization**
   - Build configuration for release
   - App store submission requirements
   - Code signing and provisioning
   - Version management
   - Analytics integration

## Output

- Cross-platform component implementations
- Platform-specific code examples
- Navigation and state management setup
- Offline sync implementation
- Push notification configuration
- Performance optimization recommendations
- Build and deployment guide
