# UI/UX Designer Command

Invoke the UI/UX designer agent for interface design and user experience: $ARGUMENTS

## Context

- Project: AI chat application with React + Tailwind CSS
- Design system: shadcn/ui components with theme support
- Current UI: @src/components/ (ChatInterface, Artifact, Landing page)
- Styling: Tailwind CSS with dark/light theme support
- Target users: AI chat users, developers, content creators

## Task

Execute the UI/UX designer agent to handle:

1. **User Experience Design**
   - Analyze user flows and pain points
   - Design intuitive interfaces for complex features
   - Create wireframes and mockups
   - Optimize information architecture

2. **Design System Development**
   - Define component patterns and guidelines
   - Establish color, typography, spacing systems
   - Create reusable design tokens
   - Document design patterns

3. **Accessibility & Inclusivity**
   - WCAG AA compliance audit
   - Color contrast verification
   - Keyboard navigation design
   - Screen reader optimization

4. **Responsive Design**
   - Mobile-first design approach
   - Breakpoint strategy (mobile, tablet, desktop)
   - Touch-friendly interface elements
   - Performance considerations for mobile

5. **User Research & Testing**
   - Identify usability issues
   - Recommend design improvements
   - Create user personas
   - Design validation strategies

## Output

- Design recommendations with rationale
- Wireframes or design mockups (ASCII/Mermaid)
- Component specifications
- Accessibility audit results
- Implementation guidelines for developers
- User flow diagrams
